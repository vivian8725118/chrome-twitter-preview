# 隐私政策

最后更新日期：2025-11-09

感谢您使用Twitter Card Preview。我们非常重视您的隐私与数据安全。本隐私政策旨在说明我们如何处理您在使用本扩展时产生的信息。

## 我们收集的数据

- 本扩展仅在 **用户主动点击扩展图标或相关按钮** 时，读取当前浏览网页的文本内容，用于提供（填写实际功能，例如“生成摘要”“翻译文本”）。
- 本扩展 **不会收集、存储或传输** 您的个人身份信息、支付信息、聊天记录、位置数据等敏感信息。

## 数据的使用方式

- 所有处理均在本地浏览器中完成。我们不会将读取到的网页内容上传至任何服务器，也不会与第三方共享。
- 读取到的网页内容只用于完成当次功能，处理完毕后立即丢弃。

## 数据的存储

- 本扩展仅使用 `chrome.storage` 保存必要的设置项（如语言偏好、是否启用某个功能等），不会保存任何网页内容或用户个人数据。
- 存储的数据仅保存在本地，可以随时在扩展设置中清除或通过卸载本扩展删除。

## 第三方服务

- 本扩展不集成任何第三方统计、广告或追踪服务。

## 权限说明

- `activeTab` / `scripting`：用于在用户授权后读取当前标签页内容并执行脚本。
- `storage`：用于存储用户的扩展偏好设置。
- （如果还有其他权限，逐条说明用途。）

## 数据共享

- 我们不会将用户数据出售、出租或披露给第三方。

## 儿童隐私

- 本扩展面向一般用户，不以 13 岁以下儿童为目标用户，也不会有意收集儿童信息。
